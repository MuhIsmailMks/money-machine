Install Phantom Wallet and Fill with SOL

Download and install Phantom Wallet from phantom.app. Create a new wallet or import one if you already have an account. Fill Phantom Wallet with SOL: You can buy SOL directly within Phantom Wallet via MoonPay or Ramp, or transfer from an exchange like Binance, Coinbase, or Kraken to your Phantom wallet address.

Open DexScreen to
Monitor Tokens on Solana

Visit DexScreen and select the Solana network. Search for the token you want to buy, such as SOL or another token running on the Solana network. DexScreen will show you relevant markets for that token on various DEXs (Decentralized Exchanges) such as Raydium or Orca.

Connect Phantom Wallet to DEX

Once you find the token you want to buy on DexScreen, it will direct you to the appropriate DEX, such as Raydium or Orca. Click Connect Wallet on DEX and select Phantom Wallet to connect it.

Swap SOL for
